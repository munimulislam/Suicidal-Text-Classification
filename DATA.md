# Dataset Access Instructions

## Kaggle SuicideWatch Dataset

- Freely available at: https://www.kaggle.com/datasets/nikhileswarkomati/suicide-watch
- Download `suicide_watch.csv` and place in `data/kaggle/`

## SWMH Dataset (Restricted Access)

- Available at: https://zenodo.org/records/6476179
- This dataset requires a formal access request using your institutional email
- Once approved, download and place files in `data/swmh/`
- Note: Per the Data Access Agreement, this dataset cannot be redistributed
